pyLandStats
===============================

Overview
--------

Open-source Python library to compute landscape metrics

Installation / Usage
--------------------

To install use pip:

    $ pip install pylandstats


Or clone the repo:

    $ git clone https://github.com/martibosch/pylandstats.git
    $ python setup.py install
